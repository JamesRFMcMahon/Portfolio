Some of My Projects

Employees Database PHP Application

A web application I built using PHP for my Web Application Programming course that allows for the viewing, searching, creating, updating, and deleting of employee records from the MySQL Employees sample database.

The application included a login mechanism, in which only authorized users could enter the site and view the data. Furthermore, only higher-level users could update and delete employee records. All passwords were encrypted in the database.
